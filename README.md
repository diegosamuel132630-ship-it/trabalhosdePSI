# Sistema de Gestão Militar

Projeto desenvolvido em Python para gerenciar entidades militares através de um menu interativo no terminal. O sistema permite criar, listar, editar e remover informações, além de guardar os dados automaticamente usando persistência.

## Funcionalidades

- Criar soldados
- Listar soldados
- Procurar soldados por ID
- Atualizar informações
- Remover soldados
- Gerenciar estruturas
- Guardar dados automaticamente
- Carregar dados ao iniciar o sistema
- Menu interativo no terminal

## Tecnologias utilizadas

- Python
- JSON
- Programação modular
- Persistência de dados

projeto/

## Como executar

### 1. Instalar Python
Baixe e instale o Python no computador.

### 2. Executar o projeto

Abra o terminal na pasta do projeto e execute:

```bash
python main.py
```
